画像投稿力の落ちない唯一のTwitterクライアント。
すべてのOSのユーザが画像投稿力の落ちないTwitterクライアントを使うことが出来るのさ。
そう、そーめん大陸ならね

そーめん大陸に画像アップロードを機能を追加しました。
画像ファイルをドラッグ&ドロップすることで、ついっぷるフォトにアップロードできます。
(複数ファイルも一括でアップロードできる。そう、薬味付きそーめん大陸ならね。)
